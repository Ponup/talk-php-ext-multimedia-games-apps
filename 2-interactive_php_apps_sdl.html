Installing PHP SDL

  • Using PECL
  • Compiling the source code
  • Using a vagrant VM

Installing PHP SDL

Using PECL command

		pecl install sdl-beta

Add extension=sdl.so to your php.ini

Installing PHP SDL

Compiling the source code

		git clone git@github.com:phpsdl/extension.git phpsdl-extension
		cd phpsdl-extension
		phpize
		./configure
		make
		sudo make install

Add extension=sdl.so to your php.ini

Installing PHP SDL

Using a vagrant VM

		git clone git@github.com:phpsdl/vagrant-machine.git phpsdl-vm
		cd phpsdl-vm
		vagrant up
