GLUT functions

glutInitWindowSize(w, h) glutInitWindowPosition(x, y) glutCreateWindow(name)
glutDisplayFunc(callback()) Called whenever the contents of the windows need to be redrawn Use glutPostRedisplay to manually ask GLUT to recall this display fx
glutReshapeFunc(callback(w, h)); Called when the window is resized or moved Call glViewport here to adjust your viewport
glutKeyboardFunc(callback(k, x, y)) Sets the keyboard callback for the current window glutIdleFunc(callback())
glutMouseFunc(callback(button, state, x, y)) glutMotionFunc(callback(x, y))
glutMainLoop() Enter the GLUT processing loop and never return glutPostRedisplay() Marks the current windows as needing to be redisplayed
